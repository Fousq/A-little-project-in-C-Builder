Добро пожаловать в программу "ПС РЫНОК ТРУДА".
Данная программа была созданная с целью, помощи поиска работы для безработных, а также для размещения данных для приёма на работу от работадателей. Для эксплотирования программы, должен быть создан альянс "Registrated". Для изучения подробностей о том как разместить или найти работу пройдите в следующие разделы: "Войти", для поиска; "Регистрация", а затем в раздел "Работадатель" для размещение вакансии.
